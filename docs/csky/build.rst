==========================
编译
==========================

------------
推荐环境
------------

推荐使用以下操作系统版本：

* ubuntu 18.04 64位
* windows 7 64位
* windows 10 64位

---------------------
获取可执行程序
---------------------

XuanTie QEMU 可执行程序有以下几种获取方式：

1. CDS安装后，以windows默认安装为例，可以在D:\\C-Sky\\CDS\\qemu下找到。
2. CDK安装后，以windows默认安装为例，可以在D:\\C-Sky\\CDK\\qemu下找到。
3. 从occ.t-head.cn芯片开放社区获取。
4. 从 T-HEAD 官方获取其他可执行版本。
